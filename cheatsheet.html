Selectors

Selector Desription
* selects every element
section all selection tags
h1, p all h1 and p tags
section p all p tags within a section
.classname all taks with a class selected name
#id all all elements inside an id item
p#classname all p elements with the class name
a:hover ancor tags hover state
a:visited ancor tags visited

Specificity Calculation

ID Column

For each ID #example in a matching selector, add 1-0-0 to the weight value.

CLASS Column

For each class selector .example or attribute selector :hover ect. in a matching selector, add 0-1-0 to the weight value.

TYPE Column

For each type selector p, h1, td, and pseudo-element ::before in a matching selector, add 0-0-1 to the weight value.

No value

The universal selector, add 0-0-0 to the weight value.

Source: MDN web docs - Selector weight categories
